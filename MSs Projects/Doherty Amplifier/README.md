# Description
Doherty amplifier is a power amplifier which introduce higher efficiency than the conventional PAs. In a convential PA, the efficiency drops at the power back-off since for a lower power the RF output voltage swing drops with respect to the supply voltage, thus dropping the efficiency. The main idea of the Doherty amplifier is to use two amplifiers, one main and one peak. The main amplifier and its loading are designed in a way that at the power back-off its voltage swing is maximized. In that case for a greater power the output of the amplifier would clip, therefore we introduce a peak amplifier in parallel.
