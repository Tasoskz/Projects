The objective of this project is to achieve the specifications for a common source amplifier with capacitive feedback with a deviation less than 10% using a systematic approach. Therefore, a matlab code was developed based in the gm/ID method. The report is short and it desrcibes analytically the approach. For the matlab and LTspice files (the last was used for verification) you can visit [here](https://drive.google.com/file/d/1ebeHDG1Og1V3YWnm1QHYEBDSaw96iei-/view?usp=sharing).
