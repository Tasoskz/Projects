This is an s-domain and t-doamin simulation for a digital PLL with the following specifications:  
| Parameter | Value |
|-----------|-------|
| Reference (REF) frequency | 100 MHz |
| REF phase noise | -155 dBc/Hz (over entire spectrum) |
| DCO phase noise | -120 dBc/Hz @ 1 MHz offset (20 dB/dec region) |
| DCO type | LC Oscillator (no upconverted flicker noise) |
| Central frequency | 2.4 GHz |
| TDC resolution | 2 ps |

The overall objective is the s-domain and t-domain modelling of the ADPLL to obtain and validate the performance of a frequency synthesizer 
for wireless applications. However, I want to stress that the time domain modelling remains unfinished.
